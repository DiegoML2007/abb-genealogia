# Proyecto: Árbol Binario de Búsqueda (ABB) - Árbol Genealógico

Este proyecto modela la genealogía de una civilización antigua mediante un Árbol Binario de Búsqueda (ABB), permitiendo registrar relaciones de parentesco y realizar consultas eficientes.

## Estructura
- `src/` : código fuente.
- `tests/` : pruebas unitarias.
- `docs/` : evidencia de colaboración.
